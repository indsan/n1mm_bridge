# yb0sps_bridge

UDP -> HTTP bridge for N1MM and JTDX. Builds Windows EXE (32-bit & 64-bit) using GitHub Actions.

Place `config.ini` next to the EXE and run.

## Release
After push to `main`, Actions will generate `yb0sps_bridge_win64.exe` and `yb0sps_bridge_win32.exe`
and attach them to a release.
